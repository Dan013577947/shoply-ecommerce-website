Shoply is a React + TypeScript e-commerce + TailwindCSS website that lets users browse products, add items to a cart, and manage their shopping experience in real-time.


### Access the Website:                
https://dan013577947.github.io/shoply-ecommerce-website/

## Key Features
- Product listing with data fetched from a public REST API
- Add/remove items and dynamic cart totals
- Persistent cart state using localStorage
- Client-side routing with React Router
- Live deployment on GitHub Pages for easy access

## Technologies & Skills
- Frontend Framework: ReactJS
- Language: Typescript
- Styling & Structure: HTML5 | TailwindCSS
- Routing and State Management: React Router, useState, useEffect, useLocation
- API Integration: REST API (GET & POST) using Axios
- Data Persistence: localStorage for cart state
- Version Control & Hosting: Git | GitHub Pages
- Development Tools: VS Code, npm
